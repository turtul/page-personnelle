JEU DE YAMS

Le jeu de Yams est un jeu de dés bien connu, ce projet reprend donc toutes les règles du Yams pour y jouer dans un terminal.

Ce projet s'inscrit dans le cadre d'un projet du premier semestre du BUT. Ce jeu a été développé en C et il est jouable entièrement dans une interfaces en lignes de commandes. Dans les règles originales du Yams il est possible de jouer à plus de deux joueurs mais dans ce projet seul deux joueurs peuvent participer.

Pour réaliser ce projet j'ai commencé par faire le maquettage de l'affichage des dés puis de la feuille de marque, ensuite j'ai conçu en pseudocode le corps principal du programme. J'ai ensuite codé en C toutes les fonctions et toutes les procédures pour faire fonctionner le jeu. Enfin j'ai testé formellement chaque entrée utilisateur à l'aide d'un cahier de tests. Pour finir j'ai générer une documentation Doxygen basée sur les commentaires du code sources. Celle-ci répertorie toutes les fonctions et procédures du programmes accompagnées d'un petit texte explicatif.
